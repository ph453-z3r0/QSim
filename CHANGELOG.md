# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Comprehensive quantum circuit analysis with state vectors, probabilities, and entanglement metrics
- Support for 9 export formats: PNG, SVG, OpenQASM, Qiskit Python, Cirq Python, JSON, PDF, LaTeX, JavaScript
- Python API for programmatic access
- Command-line interface with export and analyze commands
- Comprehensive analysis reports with tabbed interface (Results, State, Analysis)
- Probability histograms with configurable binning
- State vector tables with sortable columns
- Bloch sphere representation for single qubits
- Entanglement heatmaps and metrics
- Phase diagrams and amplitude distribution plots
- Tauri integration guide
- Complete documentation (README, API_USAGE, CLI_USAGE, QUICK_START)
